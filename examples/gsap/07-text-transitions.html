Text Transitions

Smoothly morph between different text content

⚠️ GSAP Required: Text transitions are optional GSAP-specific features. CharWrapper core works with any animation library. To use transitions, include GSAP: <script src="gsap.min.js"></script>

1. Smart Transition

Intelligently matches characters between old and new text, reusing matching characters and only animating what changed. Best for similar text.

Hello World
Strategy: smart - Reuses matching characters, minimizes DOM changes

2. Sequential Transition

Removes all characters first, then adds new ones. Clean and simple - best for completely different text.

First Text
Strategy: sequential - Remove all, then add all

3. Real-World: Counter

Practical example: animated counter that transitions smoothly between numbers.

4. Chained Transitions

Chain multiple transitions sequentially using GSAP timeline callbacks.

Watch This
